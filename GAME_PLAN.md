# D&D Virtual Table — Полный план разработки

> Источник правил: https://dnd.su/  
> При реализации любой механики — получать данные с этого сайта через WebFetch.

---

## Обзор архитектуры игры

```
Персонаж ──────────────────────────────────────────────────────────┐
  Раса + Класс + Предыстория + Черты + Умения + Заклинания + Лвл   │
                                                                    ▼
Снаряжение ──────────────────────────────────── Боевая сессия ─────┐
  Оружие, доспехи, предметы, инвентарь           Ход, Действия,    │
                                                  Атаки, Урон      │
                                                                    ▼
Монстры / НПС ───────────────────────── Карта и Окружение ─────────┐
  Стат-блоки, CR, способности          Сетка, Туман войны, AoE     │
                                                                    ▼
Мастер-инструменты ──────────────── Прогрессия ────────────────────┐
  Сцены, Энкаунтеры, Секреты         XP, Уровни, Отдых             │
                                                                    ▼
                         Игровой интерфейс
                    Чат, Лог, Листы персонажей
```

## Тема 1: Персонаж — Основа

**Что готово:** базовая модель (имя, раса, класс, уровень, 6 характеристик, текстовое снаряжение/предыстория), форма создания, шаблоны классов.

### 1.1 Расы
- Заполнить БД расами из https://dnd.su/races/
- Каждая раса: бонусы к характеристикам, особые черты, скорость, тип зрения, языки, размер
- Расы: Человек, Эльф, Дварф, Полурослик, Гном, Полуэльф, Полуорк, Тифлинг, Драконорождённый + суб-расы
- При создании персонажа — выбор расы с описанием и автоматическим применением бонусов

### 1.2 Классы (полная прогрессия)
- Таблица прогрессии 1–20 для каждого из 13 классов: https://dnd.su/class/
- Кость жизни, профиции, спасброски, начальные навыки
- Способности по уровням: когда открываются, сколько использований
- Подклассы (Архетипы): выбор на 1, 2 или 3 уровне в зависимости от класса
- Бонус мастерства (Proficiency Bonus) по уровням

### 1.3 Предыстории
- Заполнить из https://dnd.su/backgrounds/
- Каждая предыстория: 2 навыка, инструменты/языки, стартовое снаряжение, черты характера
- Примеры: Благородный, Солдат, Преступник, Народный герой, Мудрец и др.

### 1.4 Умения (Skills)
- 18 умений с привязкой к характеристике:
  - СИЛ: Атлетика
  - ЛОВ: Акробатика, Ловкость рук, Скрытность
  - ИНТ: Анализ, История, Магия, Природа, Религия
  - МДР: Внимательность, Выживание, Медицина, Уход за животными
  - ХАР: Выступление, Запугивание, Обман, Убеждение
- Владение навыком: обычное (+Бонус мастерства), экспертиза (×2)
- Пассивная проверка: 10 + модификатор

### 1.5 Черты (Feats)
- База черт из https://dnd.su/feats/
- Черты заменяют Увеличение характеристик (ASI) на определённых уровнях
- Каждая черта: название, требования, эффект
- Примеры: Боевой заклинатель, Меткий стрелок, Крепкий, Мастер большого оружия

### 1.6 Лист персонажа (полный)
- Полный экран просмотра: все 6 хар-ик + модификаторы, HP/макс HP, КБ, Инициатива, Скорость
- Спасброски с отметкой владения
- Все 18 умений с отметками владения/экспертизы
- Секция снаряжения, заклинаний, черт

## Тема 2: Прогрессия — Уровни и Развитие

**Что готово:** поле `level` в модели Character.

### 2.1 Система опыта (XP)
- Пороговые значения XP для уровней 1–20 (согласно PHB)
- Начисление XP после боя (по CR монстров)
- Автоматическое обнаружение левел-апа
- Milestone leveling как альтернатива (уровень дают вручную)

### 2.2 Повышение уровня (Level Up)
- UI для level-up: выбор новых способностей, HP (бросок или среднее), ASI/Черта
- Разблокировка подкласса на нужном уровне
- Пересчёт Бонуса мастерства

### 2.3 Отдых (Rest)
- Короткий отдых (1 час): восстановление костей жизни (добровольно)
- Долгий отдых (8 часов): полное восстановление HP, ячеек заклинаний
- Механики классовых ресурсов при отдыхе (Ярость варвара, Ячейки колдуна и т.д.)

### 2.4 Пассивные характеристики
- Бонус мастерства: +2 (1–4 ур.), +3 (5–8), +4 (9–12), +5 (13–16), +6 (17–20)
- Инициатива: модификатор Ловкости
- КБ без доспехов: 10 + мод. Лов. (для большинства классов)
- HP: кость жизни + мод. Тел. за каждый уровень

## Тема 3: Заклинания — Магическая система

**Что готово:** ничего.

### 3.1 Модель заклинания
- Поля: название, уровень (0–9), школа магии, время накладывания, дистанция, компоненты (В/С/М), длительность, концентрация, ритуал, описание, повышение уровнем
- Школы: Воплощение, Вызов, Иллюзия, Некромантия, Ограждение, Очарование, Преобразование, Прорицание
- Привязка: какие классы могут использовать
- Источник: https://dnd.su/spells/

### 3.2 Ячейки заклинаний (Spell Slots)
- Таблица ячеек для каждого класса по уровням
- Классы-заклинатели: Волшебник, Жрец, Бард, Друид, Чародей, Паладин, Следопыт, Колдун
- Восстановление: Долгий отдых (большинство), Короткий отдых (Колдун)
- Только одно заклинание концентрации одновременно

### 3.3 Известные/Подготовленные заклинания
- Волшебник: спеллбук + подготовка
- Жрец/Друид: список доступных + подготовка по ХАР/МДР
- Бард/Колдун/Чародей: фиксированный список известных

### 3.4 Колдовство в интерфейсе
- Спеллбук-панель в GameRoom
- Выбор ячейки для кастинга
- Отображение концентрации (предупреждение при прерывании)
- Кастинг в бою: как действие / бонусное действие / реакция

### 3.5 Заговоры (Cantrips)
- Заговоры не расходуют ячейки
- Урон заговоров растёт с уровнем (5, 11, 17 уровни)

## Тема 4: Снаряжение — Предметы и Инвентарь

**Что готово:** текстовое поле `equipment`.

### 4.1 Категории предметов
- Оружие (простое/воинское, рукопашное/дальнобойное)
- Доспехи (лёгкие, средние, тяжёлые, щиты)
- Снаряжение (инструменты, расходники, контейнеры)
- Магические предметы (по редкости: обычный → легендарный)

### 4.2 Оружие
- Параметры: кость урона, тип урона (рубящий/колющий/дробящий), свойства
- Свойства: Лёгкое, Тяжёлое, Двуручное, Метательное, Фехтовальное, Досягаемость, Дальнобойное
- Привязка к характеристике: СИЛ или ЛОВ (фехтовальное)

### 4.3 Доспехи
- Лёгкие: КБ = база + мод. Лов. (полный)
- Средние: КБ = база + мод. Лов. (макс +2)
- Тяжёлые: КБ = фиксированное значение, требования по СИЛ
- Щит: +2 КБ
- Владение доспехами по классу

### 4.4 Инвентарь персонажа
- Список предметов с количеством и весом
- Экипированные слоты: оружие в руках, доспех, плащ, кольца и т.д.
- Нагрузка: СИЛ × 15 фунтов (опционально)
- Валюта: ЗМ, СМ, ММ

### 4.5 Стартовое снаряжение
- Автоматическая выдача по классу + предыстории
- Или выбор вручную (золото для покупки)

## Тема 5: Боевая система — Полная механика

**Что готово:** боевая сессия, инициатива, HP/КБ, базовые атака/урон/лечение.

### 5.1 Управление ходами
- Кнопка "Следующий ход" для мастера
- Очерёдность по инициативе с отображением для всех
- Задержка хода (Delay), Готовность (Ready)
- Счётчик раундов

### 5.2 Экономика действий
- Каждый ход: Действие + Бонусное действие + Перемещение + Реакция (вне хода)
- Трекер использованных действий на текущем ходу
- Рывок (Dash): удвоить перемещение за Действие
- Отход (Disengage): уйти без атаки по возможности

### 5.3 Бросок атаки
- Формула: d20 + модификатор хар-ки + Бонус мастерства (если владение)
- Критический удар: 20 на d20 → двойные кости урона
- Автоматический промах: 1 на d20
- Преимущество/Помеха на бросок атаки

### 5.4 Типы урона и сопротивления
- 13 типов: дробящий, колющий, рубящий, огонь, холод, молния, гром, кислота, яд, некротика, лучистый, психический, силовой
- Сопротивление: урон / 2 | Иммунитет: 0 | Уязвимость: урон × 2

### 5.5 Спасброски
- 6 спасбросков по характеристикам
- d20 + модификатор + Бонус мастерства (если владение классом)
- DC: 8 + Бонус мастерства + модификатор хар-ки заклинателя/способности

### 5.6 Состояния (Conditions)
- Полный список: Ослеплён, Очарован, Оглушён, Испуган, Схвачен, Недееспособен, Невидим, Отравлен, Лежит, Обездвижен, Оглушён, Без сознания
- Каждое состояние: механический эффект в бою
- Визуальные иконки на токене карты
- Снятие вручную или автоматически по условию

### 5.7 Смерть и спасброски от смерти
- При 0 HP → без сознания, начало отсчёта
- Начало хода: d20 (10+ = успех, 9− = провал)
- 3 успеха → стабилизация | 3 провала → смерть
- Крит-провал (1) = 2 провала | Крит-успех (20) = 1 HP

### 5.8 Атаки по возможности (Opportunity Attacks)
- Триггер: существо покидает досягаемость без Рывка/Отхода
- Реакция: одна атака против уходящего

## Тема 6: Монстры и НПС

**Что готово:** токены с именем и изображением.

### 6.1 Модель монстра/НПС
- Стат-блок: 6 характеристик, HP (кость жизни × кол-во), КБ (тип доспеха), скорость
- Рейтинг опасности (CR) и награда XP
- Сопротивления, иммунитеты, уязвимости к типам урона
- Спасброски, навыки с бонусами; чувства (тёмное зрение, пассивное восприятие); языки

### 6.2 Действия монстра
- Список атак и способностей с описанием
- Мультиатака (Multiattack)
- Легендарные действия (Legendary Actions) — 3 в конце чужого хода
- Действия логова (Lair Actions) — на инициативе 20
- Реакции

### 6.3 Бестиарий
- Загрузка монстров из https://dnd.su/bestiary/
- Поиск по имени, CR, типу существа
- Предпросмотр стат-блока
- Фильтрация для мастера при построении энкаунтера

### 6.4 Добавление монстра в бой
- Мастер выбирает монстра из бестиария → появляется токен на карте
- HP: среднее из стат-блока или бросок
- НПС в очереди инициативы, управляется мастером

## Тема 7: Карта — Расширенные возможности

**Что готово:** загрузка изображения, токены с drag-and-drop, zoom/pan, декоративная сетка.

### 7.1 Функциональная сетка
- Настраиваемый размер ячейки (5 футов по умолчанию)
- Привязка токенов к ячейкам (snap to grid)
- Размер существа: Маленькое/Среднее (1×1), Большое (2×2), Огромное (3×3)

### 7.2 Туман войны (Fog of War)
- Мастер открывает/закрывает области карты
- Игроки видят только открытые области
- DM видит всё; игроки — только разведанное

### 7.3 Инструменты измерения
- Линия расстояния между двумя точками (в клетках / футах)
- Отображение дальности оружия/заклинания

### 7.4 Шаблоны зон действия (AoE)
- Окружность (Fireball), Конус (Burning Hands), Линия (Lightning Bolt), Куб (Thunderwave)
- Накладываются на карту, видны всем участникам

### 7.5 Несколько карт на сессию
- Разные этажи/локации в рамках одной игры
- Быстрое переключение, перенос токенов

### 7.6 Статусы на токене
- Иконки состояний над токеном
- HP-бар (виден мастеру, опционально — игрокам)
- Имя и инициатива токена

## Тема 8: Мастер-инструменты

**Что готово:** отправка описания сцены, загрузка карты, заглушка панели сценария.

### 8.1 Построитель энкаунтеров
- Выбор монстров из бестиария → добавление на карту
- Анализатор сложности: Easy / Medium / Hard / Deadly по уровням партии

### 8.2 Управление сессией
- Заметки мастера (видны только ему)
- Журнал сессии: все броски, атаки, события

### 8.3 Секретные броски
- Мастер бросает кубик тайно; игроки видят только "Мастер бросил кубик"

### 8.4 Раздача наград
- Суммарный XP за убитых монстров по CR
- Распределение XP между игроками
- Выдача добычи (золото, предметы)

### 8.5 Случайные таблицы
- Генератор случайных событий (интеграция с https://dnd.su/)
- Таблицы встреч по типу местности

## Тема 9: Игровой интерфейс и UX

**Что готово:** GameRoom с панелями, лобби, список игроков, бросок кубиков.

### 9.1 Полный лист персонажа в игре
- Раскрываемая панель: все хар-ки, умения, спасброски
- Текущий HP / Макс HP / Временные HP
- Ячейки заклинаний с отметками использованных

### 9.2 Книга заклинаний
- Панель со списком заклинаний персонажа
- Фильтр по уровню, школе, готовности
- Быстрый каст прямо из книги (в бою)

### 9.3 Панель инвентаря
- Список предметов с иконками
- Экипировка / снятие предмета
- Использование расходуемых предметов

### 9.4 Игровой лог / Чат
- Лента событий: броски, атаки, описания сцен
- Чат (in-character / out-of-character)
- Подсветка критических результатов

### 9.5 Трекер боя
- Список инициативы с HP-барами, виден всем
- Подсветка активного хода
- Счётчик раундов

## Тема 10: База данных игровых данных

**Что готово:** шаблоны классов (hardcoded в Python dict).

### 10.1 Источник данных
- Все игровые данные — с https://dnd.su/ через WebFetch
- Seed-скрипты для заполнения БД
- Отдельные таблицы: `races`, `class_features`, `backgrounds`, `spells`, `feats`, `weapons`, `armors`, `monsters`

### 10.2 Таблицы для заполнения

| Таблица | Источник | Приоритет |
|---------|----------|-----------|
| `races` | dnd.su/races/ | Высокий |
| `class_features` | dnd.su/class/* | Высокий |
| `backgrounds` | dnd.su/backgrounds/ | Средний |
| `spells` | dnd.su/spells/ | Высокий |
| `feats` | dnd.su/feats/ | Средний |
| `weapons` | dnd.su / PHB | Высокий |
| `armors` | dnd.su / PHB | Высокий |
| `monsters` | dnd.su/bestiary/ | Средний |

### 10.3 API для игровых данных
- `GET /api/data/races` — список рас
- `GET /api/data/classes/{name}/features` — способности класса по уровням
- `GET /api/data/spells` — список заклинаний с фильтрацией
- `GET /api/data/monsters` — бестиарий с поиском
- `GET /api/data/items` — снаряжение
