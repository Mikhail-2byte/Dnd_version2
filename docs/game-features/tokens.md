# Токены на карте

Токены представляют персонажей и монстров на игровой карте.

---

## Что такое токен

Токен — это визуальный маркер на карте, представляющий:
- Игрового персонажа (PC)
- Монстра или NPC
- Любой объект

Токен отображается как кружок с:
- Именем персонажа
- Инициалами
- Индикатором HP (опционально)
- Индикатором статуса (dead, unconscious)

---

## Создание токена

### Через UI

1. В игровой комнате нажмите на персонажа в списке
2. Перетащите персонажа на карту
3. Токен появится в указанной позиции

### Через Socket

```typescript
socket.emit("token:create", {
  game_id: 1,
  character_id: 3,
  name: "Goblin",
  x: 50,
  y: 50,
  size: 1  // занимает 1 клетку
});
```

Только **мастер** может создавать токены.

---

## Перемещение токена

### Drag & Drop

Перетащите токен мышью на нужную позицию.

### Через Socket

```typescript
socket.emit("token:move", {
  game_id: 1,
  token_id: 5,
  x: 100,
  y: 200
});
```

### Ограничения

| Роль | Может перемещать |
|------|-------------------|
| Master | Любые токены |
| Player | Только свои токены |

Синхронизация происходит мгновенно для всех игроков.

---

## Удаление токена

### Через UI

1. Кликните на токен
2. В появившемся меню нажмите "Удалить"

### Через Socket

```typescript
socket.emit("token:delete", {
  game_id: 1,
  token_id: 5
});
```

Только **мастер** может удалять токены.

---

## Обновление HP

```typescript
socket.emit("token:update_hp", {
  game_id: 1,
  token_id: 5,
  hp: 15,     // текущее HP
  max_hp: 20  // максимальное HP
});
```

HP отображается на токене (текстом или цветом).

---

## Статусы токена

| Статус | Описание |
|--------|----------|
| `active` | Персонаж в норме |
| `unconscious` | Без сознания (0 HP) |
| `dead` | Мёртв |

Смена статуса через меню токена или автоматически при HP = 0.

---

## События

### Сервер → Клиент

```typescript
// Токен перемещён
socket.on("token:moved", ({ token_id, x, y, moved_by }) => {})

// Токен создан
socket.on("token:created", ({ token }) => {})

// Токен удалён
socket.on("token:deleted", ({ token_id }) => {})

// Токен обновлён
socket.on("token:updated", ({ token_id, hp, status }) => {})
```

---

## Размер токена

Размер указывается в клетках:
- `1` — Small/Medium (1 клетка)
- `2` — Large (2x2)
- `3` — Huge (3x3)
- `4` — Gargantuan (4x4)

Визуально токен занимает соответствующую область на карте.
