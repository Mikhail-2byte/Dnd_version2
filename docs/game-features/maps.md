# Управление картами

Карты — это изображения, на которых размещаются токены.

---

## Загрузка карты

### Через UI

1. В игровой комнате нажмите "Загрузить карту"
2. Выберите файл изображения (jpg, png, webp)
3. Карта загрузится и отобразится

### Через HTTP

```bash
curl -X POST http://localhost:8000/api/maps/upload \
  -H "Authorization: Bearer <token>" \
  -F "file=@map.jpg"
```

**Ограничения:**
- Максимальный размер: 10MB
- Форматы: jpg, png, webp

### Ответ

```json
{
  "url": "/uploads/maps/uuid.jpg"
}
```

---

## Отображение карты

### В игре

Карта отображается в центральной области экрана.

### Zoom (масштаб)

| Действие | Управление |
|----------|------------|
| Увеличить | Колёсико мыши вверх |
| Уменьшить | Колёсико мыши вниз |
| Сбросить | Двойной клик |

### Pan (перемещение)

| Действие | Управление |
|----------|------------|
| Переместить | Правая кнопка мыши + drag |
| Сбросить | Кнопка "Сбросить вид" |

### Сетка

Сетка отображается поверх карты (опционально):
- Размер клетки: 70px (по умолчанию)
- Привязка токенов к сетке (опционально)

---

## Размеры карты

По умолчанию используется размер загруженного изображения.

Для работы с токенами используется система координат:
- `x` — горизонтальная позиция (слева)
- `y` — вертикальная позиция (сверху)

---

## Состояние карты в игре

```json
{
  "game": {
    "id": 1,
    "map_url": "/uploads/maps/uuid.jpg"
  },
  "tokens": [
    {
      "id": 1,
      "name": "Aragorn",
      "x": 350,
      "y": 200,
      "size": 1
    }
  ]
}
```

---

## Ограничения

- Загружать карты может только **мастер**
- Размер файла до 10MB
